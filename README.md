# EDGE03 Website

Premium business website starter for EDGE03 LLC.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Vercel-ready build

## Run Locally

```bash
npm install
npm run dev
```

Production preview:

```bash
npm run build
npm run start
```

## Editable Content

Most placeholder copy and repeatable content lives in `lib/content.ts`.

Primary sections and routes live in:

- `app/page.tsx`
- `app/services/page.tsx`
- `app/process/page.tsx`
- `app/about/page.tsx`
- `app/contact/page.tsx`
- `app/industries/page.tsx`
- `app/case-studies/page.tsx`
- `app/maintenance-plans/page.tsx`

Reusable layout and placeholder pieces live in `components/`.

## Future Assets

Logo, image, client logo, testimonial, stat, and case study placeholders are intentionally marked in the code so real assets can be dropped in later without rebuilding the layout.
